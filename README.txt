[作品紹介]
作品名：Bounceback Blitz
ジャンル：横スクロールアクション
制作期間：2023/10/02〜2023/10/27

風のクロノアのマップ設計×LIBLADEの斬撃システムを合体させたゲームです


[作品概要]
・ツインスティックのアクションゲーム
・スティックを弾いた方向に斬撃が発生し、敵の弾を跳ね返すことが出来る
・クロノアのように奥行きのあるマップに対しても敵が出現します。
　跳ね返す方向を場に合わせて攻略しよう

[実装内容]
※現段階はα版となります
基本の操作となる斬撃、地形に対する移動とカメラの動きの作成

[操作方法] ※ツインスティック対応のコントローラーが必要になります
左スティック：移動
右スティック：攻撃
LB：ジャンプ
START：ポーズ
